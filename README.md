Velora is a voice-powered assistant and video-enabled web application built with Node.js, Express, and Socket.io.

Features
🎤 Voice assistant interaction
🌐 Web-based interface
📹 Real-time video communication (Socket.io)
🔐 User authentication (planned)
🛒 Shopify integration (planned)
💰 Google Ads monetization (planned)
Tech Stack
Node.js
Express.js
Socket.io
HTML / CSS / JavaScript
